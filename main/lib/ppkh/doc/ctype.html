Library: Ctype


View source.

Description

This library contains routines for (unit-based) characters.
There are two sorts of routines:
  • character checks: These routines will check it the character is of a certain type, when it isn't, the fun fails.
  • character conversions: These routines will convert the character.

    Routine Description
    isalnum This routine checks if Input Variable [ctypeinput] is alphanumeric: {0..9 A..Z a..z}.
    isalpha This routine checks if Input Variable [ctypeinput] is alphabetic: {A..Z a..z}.
    isascii This routine checks if Input Variable [ctypeinput] is an ascii-character.
    iscntrl This routine checks if Input Variable [ctypeinput] is a control-character.
    isdigit This routine checks if Input Variable [ctypeinput] is a digit, {0..9}.
    islower This routine checks if Input Variable [ctypeinput] is a lower case character, {a..z}.
    isgraph This routine checks if Input Variable [ctypeinput] is a graphical character.
    isprint This routine checks if Input Variable [ctypeinput] is a printable character.
    ispunct This routine checks if Input Variable [ctypeinput] is a punctuation character.
    isspace This routine checks if Input Variable [ctypeinput] is an empty space character.
    isupper This routine checks if Input Variable [ctypeinput] is an upper case character, {A..Z}.
    isxdigit This routine checks if Input Variable [ctypeinput] is a hexadecimal character, {0..9 a..z A..F}.
    toascii This routine converts Input Variable [ctypeinput] to an ascii character.
    tochar This routine converts Input Variable [ctypeinput] to a byte-character.
    tolower This routine converts Input Variable [ctypeinput] to a lower-case character if it's an upper-case character.
    toupper This routine converts Input Variable [ctypeinput] to an upper-case character if it's a lower-case character.
